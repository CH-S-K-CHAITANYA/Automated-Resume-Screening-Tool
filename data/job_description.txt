Data Scientist / Machine Learning Engineer

deploy machine learning solutions that power our core products.

- Perform exploratory data analysis (EDA), feature engineering, and statistical modeling
- Write clean, maintainable Python code; build data pipelines using Pandas and NumPy
- Apply NLP techniques for text classification, sentiment analysis, and entity extraction

- Collaborate with data engineers and software engineers
- Build data visualizations and communicate insights to non-technical stakeholders
- Deploy models as REST APIs using Flask or FastAPI
- Containerize services with Docker; work with AWS cloud infrastructure
- Version control all code with Git
